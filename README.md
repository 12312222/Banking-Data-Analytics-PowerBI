# 🏦 Banking Data Analytics Dashboard (Power BI)

## 📌 Project Overview
This project develops an interactive Power BI dashboard to analyze banking customer behavior, transaction activities, revenue performance, and business trends. The dashboard helps identify high-value customer segments, monitor transaction patterns, and support data-driven decision-making.

---

## 🎯 Business Objectives

- Analyze customer segmentation and transaction behavior.
- Identify the most active customer groups.
- Monitor transaction channels and product usage.
- Evaluate revenue generated from banking fees.
- Detect trends and emerging patterns over time.

---

## 🛠️ Tools & Technologies

- Power BI
- Power Query
- DAX
- Excel

---

## 📊 Key Performance Indicators (KPIs)

- Total Transactions
- Total Transaction Amount
- Total Fees Revenue
- Average Customer Score
- Number of Customers
- Average Monthly Income

---

## 📈 Dashboard Pages

### 1. Customer Insights & Segmentation
- Customer activity by income segment.
- Product popularity across segments.
- Relationship between customer score and transaction behavior.

### 2. Transaction Behavior Analysis
- Transaction volume and value by transaction type.
- Channel usage analysis (Mobile, ATM, Branch, Online).
- Branch and city performance analysis.

### 3. Revenue & Cost Analysis
- Revenue generated from different fee types.
- Customer fee burden analysis.
- Revenue sources and friction points.

### 4. Trend & Performance Analysis
- Transaction trends over time.
- Offer effectiveness analysis.
- Emerging trends by customer segment and channel.

---

## 🔍 Key Insights

- High-income customers generate the highest transaction values.
- Mobile banking is the most frequently used transaction channel.
- Credit card fees contribute significantly to fee revenue.
- Transaction activity shows seasonal patterns across months.
- Premium customer segments continue to grow over time.
